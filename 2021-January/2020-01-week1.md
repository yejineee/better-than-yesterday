**잘한 점**

- 미루고 미뤘던 데일리 회고를 시작하였다.
  잘 살아본 날부터 회고를 쓰고 싶었지만, 그런 날은 없었다. 일단 회고를 써야 잘 살 수 있을 것이라고 생각을 바꾸어보았다. 오늘 계획대로 살지는 못했지만 데일리 회고를 시작하였다. 시작이 반이니 아주 잘했다.
- 자바스크립트 prototype에 대해 다시 한 번 더 제대로 공부하였다.
  Object.create가 무엇인지 찾아보다가 다시 한 번 더 프로토타입에 대해 공부해보았다. 여름에 프로토타입에 대해 공부한 것을 정리한 것을 바탕으로 공부하였다. 그 때 정리했던 것들 중 애매한 것은 지우고 다시 명확한 표현으로 바꾸었다. 나중을 위하여 배운 것을 정리해두는 것은 너무 좋은 것 같다.
- 프로그래머스 문제 풀고, 정리한 것 + 새로운 풀이로 풀어본 것
  1단계이긴 하지만, 알고리즘 풀었던 것을 정리한 것은 잘한 것 같다. 컴과사 수업에서 배운대로 Fact와 complexity를 계산한 것도 잘했다. 또, 다른 사람들의 풀이를 보고 regex로 다시 풀어본 것도 잘한 점이다. 나중에 문자열 문제를 만나게 되면 그때도 regex로 도전해보면 좋을 것 같다.
- 마이리얼트립 지원한 것
  일단 한 걸음이 중요하니깐, 이력서에 쓸 말 없다고 시무룩해있지 않고 지원한 것은 잘했다.
